# ME ENCARGO — Releases

Este repositorio aloja únicamente las **descargas públicas** de las aplicaciones
móviles de **ME ENCARGO** (NH Cargo).

No contiene código fuente. El código de las apps se mantiene en repositorios
privados; aquí solo se publican los APK firmados que genera automáticamente la
integración continua (CI).

## Descargas

Las versiones están en la sección [**Releases**](../../releases).

- **App de Choferes** — `app_drivers_nhcargo_universal.apk`
- **App de Clientes** — `app_clients_nhcargo_universal.apk`

Enlace estable a la última versión:

```
https://github.com/athendat/meencargo-releases/releases/latest/download/<archivo>.apk
```

## Verificación de integridad

Cada APK muestra su huella **SHA-256** junto al archivo en la página del release.
Compárala después de descargar para confirmar que el archivo no fue alterado.
Todos los APK están firmados con la clave de release oficial de ME ENCARGO.

## Instalación

Estos APK se instalan de forma directa (sideload) mientras completamos la
publicación en las tiendas oficiales. El dispositivo puede solicitar habilitar
la instalación de aplicaciones de orígenes desconocidos.
